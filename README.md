# Caderno Temático com NotebookLM

## Aprendizado de Inglês: do Nível Iniciante ao B1

Projeto desenvolvido como desafio da DIO utilizando o NotebookLM para aprendizagem focado no estudo da língua inglesa.

# Objetivos

Este projeto tem como objetivo utilizar o NotebookLM para:
- Organizar conteúdos sobre aprendizagem de inglês do nível A1 até B1;
- Criar um professor virtual de inglês baseado em fontes confiáveis;
- Construir um guia de estudos reutilizável.
